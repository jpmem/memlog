Windows Update Agent 初期調査に必要な資料採取手順について

Windows 端末での Windows Update に関わる初期調査では copylogs.cmd により資料採取をお願いしております。

対象

Windows Update に問題が発生した端末。
Configuration Manager で更新プログラムの配布を受け取っているクライアントは除きます。Configuration Manager クライアントの場合は、Configuration Manager 初期調査に必要な資料採取手順について による資料採取をお願いいたします。

手順

下記 URL にアクセスし、zip ファイルをダウンロードして、展開してください。

https://github.com/microsoft-jpcssmem/WSUS/raw/master/copylogs.cmd.zip

  1. 展開先にある copylogs.cmd を採取対象端末にコピーします。
  2. copylogs.cmd を管理者権限で実行します。
  3. コマンドプロンプトが開き処理が進みます。
  4. 以下のメッセージが出力されたら適宜キーを入力しコマンドプロンプトを閉じてください。
  5. Cドライブのルート直下を開き、下記のようなファイルがあるのを確認します。
  6. 作成されたファイルを、アップロードいただけますよう、お願い致します。
    (ファイル名) WULogs-<アカウント名>-<端末名>-<日付>.ZIP

本情報の内容（添付文書、リンク先などを含む）は、作成日時点でのものであり、予告なく変更される場合があります。
