アプリケーション識別情報の採取手順について

調査対象のアプリケーションについて、下記でご案内する識別情報をご提供くださいますようお願いいたします。

作業端末

Configuration Manager コンソールが動作する端末

採取情報

  1. アプリケーションの名前
  2. アプリケーション構成情報 ID となる、「CI の固有 ID」
  3. 展開の種類の「コンテンツ ID」
  4. 展開の「パッケージ ID」、「モデル名」、「展開 ID」

採取手順

調査対象のアプリケーションについて、Configuration Manager コンソールから以下の手順で識別情報を採取し、ご提供ください。

  1. [Configuration Manager コンソール] - [ソフトウェア ライブラリ] - [概要] - [アプリケーション 管理] - [アプリケーション] と遷移します。
  2. 表示項目の列名を右クリックし、「CI の固有 ID」を追加します。
  3. 「CI の固有 ID」が表示されている状態で、対象のアプリケーションを選択し、キーボードで「Ctrl + C」を押下します。
  4. メモ帳等を起動し、「Ctrl + V」を押下すると選択した行が貼り付けられます。
  5. Configuration Manager コンソールの画面に戻り、対象のアプリケーションが選択されている状態で、画面下部「展開の種類」を選択します。
  6. 「コンテンツ ID」が表示されている状態で、対象の展開の種類を選択し、キーボードで「Ctrl + C」を押下します。
  7. メモ帳等で「Ctrl + V」を押下すると選択した行が貼り付けられます。
  8. Configuration Manager コンソールの画面に戻り、対象のアプリケーションが選択されている状態で、画面下部「展開」を選択します。
  9. 表示項目の列名を右クリックし、「パッケージ ID」、「モデル名」、「展開 ID」を追加します。
  10. 「パッケージ ID」、「モデル名」、「展開 ID」が表示されている状態で、対象の展開を選択し、キーボードで「Ctrl + C」を押下します。
  11. メモ帳等で「Ctrl + V」を押下すると選択した行が貼り付けられます。
  12. アプリケーションの名前に加えて、上記で入手した「CI の固有ID」、「コンテンツ ID」、「パッケージ ID」、「モデル名」、「展開 ID」の情報を弊社にご提供ください。
