PatchDownloader.log の採取手順について

PatchDownloader.log は、Configuration Manager コンソールや自動展開規則でソフトウェア更新プログラムをダウンロードした時の動作が記録されます。
ログは複数の場所に作成されている可能性がございますので、下記でご案内する手順で採取した PatchDownloader.log をご提供くださいますようお願いいたします。

作業端末

  • Configuration Manager コンソールが動作する端末
  • サイト サーバー

採取情報

  1. PatchDownloader.log または PatchDownloader.lo_

採取手順

  1. CD/DVD Drive 以外の認識されている全ドライブ文字を対象として、以下のコマンドを実行します。
  2. 上記コマンドにて発見された全ての PatchDownloader.log または PatchDownloader.lo_ について、フォルダーにまとめ、zip ファイルに圧縮の上、採取をお願いいたします。
